name: Never Suggest Discarding Work
description: NEVER offer "discard changes" as an option for uncommitted work. Not even as a choice.
type: feedback
---

NEVER suggest discarding uncommitted work. Not as an option. Not as a choice. Not even listed as a possibility.

**Why:** Brent was furious when "discard" appeared as option 3 for uncommitted repos during /brent-finish. Uncommitted work is someone's effort. Offering to throw it away is insulting.

**How to apply:** When presenting options for uncommitted work, the only choices are:
1. Commit WIP now
2. Leave uncommitted (will handle later)

There is no third option. Ever.
