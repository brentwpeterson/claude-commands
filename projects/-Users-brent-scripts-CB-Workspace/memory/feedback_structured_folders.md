---
name: Structured Folders Are Better
description: Always use organized folder structures. Never dump files in random locations. Ask when unsure where something goes.
type: feedback
---

Structured folders are always better than flat file dumps. 100 files in organized folders is faster to search than 10 billion random files.

**Why:** The E-E-A-T blog post was saved to the wrong folder (`Content Cucumber/Blog/` instead of `Brent Notes/Sales and Marketing/Company Websites/Content Cucumber/Blog/`). Multiple files were dumped in the CC root folder instead of subfolders. This wastes time and creates confusion.

**How to apply:**
- Before creating ANY file, check the folder structure and put it in the right place
- If you do not know where a file goes, ASK. Do not guess.
- Every company folder needs a clear structure (Website/, Sales/, Strategy/, Blog/, Social/, Emails/)
- Never dump loose files in a folder root when subfolders exist
- Read the README.md in the target folder before writing files there
