---

## Date Conditionals

```
IF: Week number is even → Add "Content in Commerce LinkedIn Live at noon" to punch list
IF: P0 event within 7 days → Add "Event prep: [event name]" to punch list
```

**How to calculate:**
```bash
WEEK_NUM=$(date +%V)
IS_EVEN=$(( WEEK_NUM % 2 ))
# IS_EVEN == 0 means even week → Content in Commerce week
```
